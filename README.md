# Minimal REST API

## Project template to fork and implement real rest api

Dependences

- [**gorilla/mux**](https://github.com/gorilla/mux) as router
- [**zerolog**](github.com/rs/zerolog/log) as logger
- Postgresql as db
- Authtentication with JWT generated at startup (see log)
